Laser target keyword related blogs and websites, for commenting, back links and finding fresh content, direct from the WordPress dashboard.

== Description ==

This plugin quickly find relevant blogs for comments, links, traffic and engagement.
It searches for blogs or websites based on keywords.
It appears under the edit page or post screen.

**Settings:** Simply activate and you will see a new meta box on the "Post" and "Page" edit screens. No other setting required! 

**Usage:** Enter a keyword, select the "type", click update keyword button, and a list of relevant articles will appear.

== Installation ==

1. Upload "**BloggBuddy**" folder to the "wp-content/**plugins**" directory.
2. Activate the plugin through the "**Plugins**" menu in WordPress.
